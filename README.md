# Wellness Tourism MLOps Pipeline

## Overview

This project implements an end-to-end MLOps pipeline for predicting whether a customer is likely to purchase a Wellness Tourism Package.

The project includes:
- Data Registration
- Data Preparation
- Model Training
- Hyperparameter Tuning using GridSearchCV
- Experiment Tracking
- Model Registration
- GitHub Actions Workflow
- Hugging Face Integration
- Streamlit Deployment

## Model

Algorithm: Random Forest Classifier

Accuracy: 91.6%
